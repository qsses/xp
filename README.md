# Windows XP 2005 Clone

一个纯 HTML/CSS/JavaScript 的 Windows XP 怀旧桌面模拟器。

## 在线演示

- [qssec1.github.io/xp](https://qssec1.github.io/xp/)
- [qsses.github.io/xp](https://qsses.github.io/xp/)

## 运行

直接打开 `index.html`，或在当前目录启动静态服务器：

```powershell
python -m http.server 4173
```

访问 `http://127.0.0.1:4173/`。

## 已实现

- 开机、登录、注销、关机和重新启动
- XP 桌面、任务栏、开始菜单、托盘时钟和右键菜单
- 可拖动、缩放、最小化和最大化的窗口
- 我的电脑、我的文档、回收站、控制面板
- 记事本、画图和可访问互联网的 Internet Explorer
- 游戏中心：扫雷、贪吃蛇、俄罗斯方块、纸牌接龙、红色警戒2
- XP 风格启动提示音、通知音和双行系统时钟
- 命令提示符、计算器和 Windows 任务管理器
- 记事本打开/保存/下载、字体与换行设置
- 画图打开图片、保存 PNG、撤销/重做和自动恢复
- 记事本、画图及回收站的本地持久化
